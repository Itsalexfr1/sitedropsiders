# ⚠️ MESSAGE DE SÉCURITÉ CHROME

## Vous voyez ce message ?

```
Warning: Don't paste code into the DevTools Console that you don't understand 
or haven't reviewed yourself. This could allow attackers to steal your identity 
or take control of your computer. Please type 'allow pasting' below and press 
Enter to allow pasting.
```

## ✅ C'EST NORMAL ! Voici quoi faire :

### ÉTAPE 1 : Débloquer le collage
1. Dans la console Chrome, **tapez exactement** :
   ```
   allow pasting
   ```
2. Appuyez sur **Entrée**

➡️ **Le message disparaît !**

---

### ÉTAPE 2 : Maintenant coller le script
1. **Collez** le script à nouveau (**Ctrl + V**)
2. Appuyez sur **Entrée**

➡️ **Le script démarre ! 🚀**

---

## 🛡️ Pourquoi ce message ?

Chrome vous protège contre les scripts malveillants. C'est une bonne chose !

**Mais notre script est sûr** car :
- ✅ Vous l'avez créé vous-même
- ✅ Il ne fait que lire des données publiques
- ✅ Il ne touche pas à vos informations personnelles
- ✅ Il télécharge juste un fichier JSON

---

## 📝 RÉSUMÉ COMPLET

1. **F12** pour ouvrir la console
2. **Onglet "Console"**
3. Tapez : `allow pasting` + **Entrée**
4. **Ctrl + V** pour coller le script
5. **Entrée** pour lancer

**C'est parti ! 🎬**
